    使用

    • 写入 <router-link to="/">内容</router-link>
    • 在页面引入(可以单独存在,不一应需要 router-link ) <router-view></router-view>
    • 用 Vue.js + Vue Router 创建单页应用,是非常简单的.使用 Vue.js 我们已经可以通过组合组建来组成应用程序,当你要把 Vue Router 添加进来,我们需要做的是,将组建(components)映射到路由(routes),然后告诉 Vue Router 在哪里渲染他们
    • 通过注入路由器,我们可以在任何组建内通过 this.$router 访问路由器,也可以通过 this.$router 访问当前路由
    • 需要注意的是,当 <router-link> 对应的路由匹配成功,将自动设置class属性值 .router-link-active ,可以在 router.js 中new Router 下加入 linkExactActiveClass:'自定义的class名字' 改变 .router-link-active 这个 class 名字
